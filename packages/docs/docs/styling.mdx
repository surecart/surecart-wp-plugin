# Styling

SureCart components use a [shadow DOM](https://developer.mozilla.org/en-US/docs/Web/Web_Components/Using_shadow_DOM) to encapsulate their styles. This provides a few advantages:

- Styles are protected from any potential style conflicts (theme styles, for example).
- If the internal HTML structure of any components change, you won't need to update any custom css.

There are two ways to make customizations to SureCart components. One way is with CSS custom properties ("CSS variables"), the other is with CSS parts ("parts").

## CSS Variables
CSS variables are great for global style changes that are shared across components. A good example of this is a border radius. Form elements, inputs, cards and buttons all share the same border radius.
To make it as easy as possible to change this, you can set the `--sc-border-radius` variables to change this globally.

Example:
```css
:root:root {
  /** Remove all border radiuses */
  --sc-border-radius-small: 0;
  --sc-border-radius-medium: 0;
  --sc-border-radius-large: 0;
  --sc-border-radius-x-large: 0;
}
```

## CSS Parts
While CSS Variables offer a high-level way to customize the SureCart, css parts offer a low-level way to customize individual components. This is done with pure CSS â€” nothing else is required.

SureCart components use a shadow DOM to encapsulate their styles. This means you can't simply target their internals with the usual CSS selectors as you are used to. Instead, components expose "parts" that can be targeted with the CSS part selector, or `::part()`.

Here's an example that modifies an input component to a more techy/brutalist style:

import DemoTabs from '../components/Demo-Tabs.js';
import ComponentDoc from '../components/Component-Doc.js';

<DemoTabs>
	<DemoTabs.Code>
		{ `
<sc-input label="Your Name" class="sc-input-style-example" placeholder="Enter your name"></sc-input>
<style>
.sc-input-style-example::part(base) {
  border: 1px solid black;
  box-shadow: 2px 2px #2dd4bf;
  border-radius: 0;
  font-family: monospace;
}
.sc-input-style-example::part(base):hover {
  border-color: #2dd4bf;
  box-shadow: 5px 5px #2dd4bf;
}
.sc-input-style-example::part(label) {
  color: black;
  font-size: 14px;
  letter-spacing: 2px;
  text-transform: uppercase;
  font-family: monospace;
}
.sc-input-style-example::part(input)::placeholder {
  letter-spacing: 2px;
  text-transform: uppercase;
}
</style>` }
	</DemoTabs.Code>
	<sc-input label="Your Name" class="sc-input-style-example" placeholder="Enter your name"></sc-input>
</DemoTabs>
