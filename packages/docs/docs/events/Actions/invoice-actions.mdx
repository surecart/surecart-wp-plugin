# Invoice Actions Reference

### checkout_engine/invoice_created

Occurs when a new invoice is created. This is typically `draft` status that but can be `paid` or `finalized` as well.

```php
do_action(
	'checkout_engine/invoice_created',
	(string) $invoice_id,
	(array) $webhook_data
)
```

### checkout_engine/invoice_finalized

Finalizes a specific invoice before it can be paid.

```php
do_action(
	'checkout_engine/invoice_finalized',
	(string) $invoice_id,
	(array) $webhook_data
)
```


### checkout_engine/invoice_made_draft

Occurs whenever an invoice is reverted to the `draft` state.

```php
do_action(
	'checkout_engine/invoice_made_draft',
	(string) $invoice_id,
	(array) $webhook_data
)
```

### checkout_engine/invoice_paid

Occurs when a new invoice has been successfully paid.

```php
do_action(
	'checkout_engine/invoice_paid',
	(string) $invoice_id,
	(array) $webhook_data
)
```


### checkout_engine/invoice_updated

Occurs when an invoice is updated. This could include changing line items, addresses, discounts and more.

```php
do_action(
	'checkout_engine/invoice_updated',
	(string) $invoice_id,
	(array) $webhook_data
)
```
