## Examples

All at once:
```php
$product = \SureCart\Models\Product::find('8ba8d60f-5277-4e6b-807c-dee8166446d5');
$product->update([
	'name' => 'New name',
	'description' => 'New description',
]);
```

Wait until attributes are updated:
```php
$product = \SureCart\Models\Product::find('8ba8d60f-5277-4e6b-807c-dee8166446d5');
$product->name = 'New name';
$product->descriptoin = 'New description';
$product->save();
```
