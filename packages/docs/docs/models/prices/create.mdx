# Create

Creates a new price.

## Attributes
import ModelBody from '../../../components/ModelBody.js';

<ModelBody path={'/api/v1/prices'} method={'post'} model={'price'} />

## Examples

```php
$saved = new CheckoutEngine\Models\Price([
	'name' => 'Price 1',
	'amount' => 1000,
	'currency' => 'usd',
	'recurring' => true, // recurring price
	'recurring_interval' => 'month', // monthly
	'recurring_interval_count' => 3 // every 3 months
])->save();
```
