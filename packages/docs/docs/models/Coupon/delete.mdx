## Examples

Delete a model by id:
```php
\SureCart\Models\Coupon::delete('8ba8d60f-5277-4e6b-807c-dee8166446d5');
```

Delete a model directly.
```php
$coupon = \SureCart\Models\Coupon::find('8ba8d60f-5277-4e6b-807c-dee8166446d5');
$coupon->delete();
```
