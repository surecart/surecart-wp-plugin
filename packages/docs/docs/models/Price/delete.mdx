## Examples

Delete a model by id:
```php
\CheckoutEngine\Models\Price::delete('8ba8d60f-5277-4e6b-807c-dee8166446d5');
```

Delete a model directly.
```php
$price = \CheckoutEngine\Models\Price::find('8ba8d60f-5277-4e6b-807c-dee8166446d5');
$price->delete();
```
