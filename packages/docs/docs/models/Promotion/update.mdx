## Examples

All at once:
```php
$promotion = \CheckoutEngine\Models\Product::find('8ba8d60f-5277-4e6b-807c-dee8166446d5');
$promotion->update([
	'code' => 'NEWCODE',
]);
```

Wait until attributes are updated:
```php
$promotion = \CheckoutEngine\Models\Product::find('8ba8d60f-5277-4e6b-807c-dee8166446d5');
$promotion->code = 'NEWCODE';
$promotion->save();
```
