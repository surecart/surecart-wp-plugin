# List
Returns a list of your prices.

## Where Parameters
import ModelParameters from '../../../components/ModelParameters.js';

<ModelParameters path={'/api/v1/prices'} method={'get'} />

## Examples

```php
\CheckoutEngine\Price::where([
	'product_ids' => [
		'8ba8d60f-5277-4e6b-807c-dee8166446d5',
		'07fd28e0-e735-4acf-86d8-4d7855055e0b'
	],
	'archived' => false
])->paginate([
	'page' => 1,
	'per_page' => 20
]);
```
