# Button

```html
<sc-button></sc-button>
```

Buttons let users take actions and make choices with a single click or tap.

import DemoTabs from '../../components/Demo-Tabs.js';
import ComponentDoc from '../../components/Component-Doc.js';

<DemoTabs>
	<DemoTabs.Code>
		{ '<sc-button type="primary">Click Me</sc-button>' }
	</DemoTabs.Code>
	<sc-button type="primary">Click Me</sc-button>
</DemoTabs>

## Type

Use the `type` property to set the button's type.

<DemoTabs>
	<DemoTabs.Code>
		{ `
<sc-button>Default</sc-button>
<sc-button type="primary">Primary</sc-button>
<sc-button type="success">Success</sc-button>
<sc-button type="info">Info</sc-button>
<sc-button type="warning">Warning</sc-button>
<sc-button type="danger">Danger</sc-button>
    ` }
	</DemoTabs.Code>
	<sc-button>Default</sc-button> <sc-button type="primary">
		Primary
	</sc-button> <sc-button type="success">Success</sc-button> <sc-button type="info">
		Info
	</sc-button> <sc-button type="warning">Warning</sc-button> <sc-button type="danger">Danger</sc-button>
</DemoTabs>
