# Button

```html
<presto-button></presto-button>
```

Buttons let users take actions and make choices with a single click or tap.

import DemoTabs from "../../components/Demo-Tabs.js";
import Tabs from "@theme/Tabs";
import TabItem from "@theme/TabItem";

<DemoTabs>
  <DemoTabs.Code>

  </DemoTabs.Code>

</DemoTabs>

Use the `type` property to set the button's type.

<presto-button type="info">Info</presto-button>
<presto-button type="warning">Warning</presto-button>
<presto-button type="danger">Danger</presto-button>
    `}
  </DemoTabs.Code>
