# Button

```html
<ce-button></ce-button>
```

Buttons let users take actions and make choices with a single click or tap.

import DemoTabs from '../../components/Demo-Tabs.js';
import ComponentDoc from '../../components/Component-Doc.js';

<DemoTabs>
	<DemoTabs.Code>
		{ '<ce-button type="primary">Click Me</ce-button>' }
	</DemoTabs.Code>
	<ce-button type="primary">Click Me</ce-button>
</DemoTabs>

## Type

Use the `type` property to set the button's type.

<DemoTabs>
	<DemoTabs.Code>
		{ `
<ce-button>Default</ce-button>
<ce-button type="primary">Primary</ce-button>
<ce-button type="success">Success</ce-button>
<ce-button type="info">Info</ce-button>
<ce-button type="warning">Warning</ce-button>
<ce-button type="danger">Danger</ce-button>
    ` }
	</DemoTabs.Code>
	<ce-button>Default</ce-button> <ce-button type="primary">
		Primary
	</ce-button> <ce-button type="success">Success</ce-button> <ce-button type="info">
		Info
	</ce-button> <ce-button type="warning">Warning</ce-button> <ce-button type="danger">Danger</ce-button>
</DemoTabs>
