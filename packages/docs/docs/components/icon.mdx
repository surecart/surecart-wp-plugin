# Icon

```html
<sc-icon name="icon-name-here"></sc-icon>
```

This component let users display an icon. SureCart comes bundled with over 287 icons courtesy of the [Feather Icons](https://feathericons.com/). Please use the names from this library for the name property of the icon.
