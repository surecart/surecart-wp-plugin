# Choice

```html
<ce-choice></ce-choice>
```

Choice buttons allow selection of a single or multiple options
from a set of predefined options. They are a good alternative to
radio buttons, checkboxes, and select menus.

:::info
Choice components must be wrapped in the `<ce-choices></ce-choices>` component.
:::

import DemoTabs from '../../components/Demo-Tabs.js';

<DemoTabs>
  <DemoTabs.Code>
{`
<ce-choices>
    <ce-choice>
        Gold Plan
        <span slot="description">$9.99, then $49.99 per month</span>
    </ce-choice>
    <ce-choice checked>
        Silver Plan
        <span slot="description">$39.99 per month</span>
    </ce-choice>
    <ce-choice>
        Bronze Plan
        <span slot="description">$9.99 per month</span>
    </ce-choice>
</ce-choices>
`}

  </DemoTabs.Code>
    <ce-choices>
        <ce-choice>
            Gold Plan
            <span slot="description">$9.99, then $49.99 per month</span>
        </ce-choice>
        <ce-choice checked>Silver Plan
            <span slot="description">$39.99 per month</span>
        </ce-choice>
        <ce-choice>Bronze Plan
            <span slot="description">$9.99 per month</span>
        </ce-choice>
    </ce-choices>
</DemoTabs>

## Checkbox Type

In addition to a radio style selector, you can also choose a checkbox. This
lets you change the behavior from selecting one option from many, to selecting
any number of options, or none at all.

<DemoTabs>
  <DemoTabs.Code>
{`
<ce-choices>
    <ce-choice type="checkbox">
        Gold Plan
        <span slot="description">$9.99, then $49.99 per month</span>
    </ce-choice>
    <ce-choice checked>
        Silver Plan
        <span slot="description">$39.99 per month</span>
    </ce-choice>
    <ce-choice type="checkbox">
        Bronze Plan
        <span slot="description">$9.99 per month</span>
    </ce-choice>
</ce-choices>
`}

  </DemoTabs.Code>
    <ce-choices>
        <ce-choice type="checkbox">
            Gold Plan
            <span slot="description">$9.99, then $49.99 per month</span>
        </ce-choice>
        <ce-choice type="checkbox" checked>Silver Plan
            <span slot="description">$39.99 per month</span>
        </ce-choice>
        <ce-choice type="checkbox">Bronze Plan
            <span slot="description">$9.99 per month</span>
        </ce-choice>
    </ce-choices>
</DemoTabs>
