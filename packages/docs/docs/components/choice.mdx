# Choice

```html
<presto-choice></presto-choice>
```

Choice buttons allow selection of a single or multiple options
from a set of predefined options. They are a good alternative to
radio buttons, checkboxes, and select menus.

:::info
Choice components must be wrapped in the `<presto-choices></presto-choices>` component.
:::

import DemoTabs from "../../components/Demo-Tabs.js";

<DemoTabs>
  <DemoTabs.Code>
{`
<presto-choices>
    <presto-choice>
        Gold Plan
        <span slot="description">$9.99, then $49.99 per month</span>
    </presto-choice>
    <presto-choice checked>
        Silver Plan
        <span slot="description">$39.99 per month</span>
    </presto-choice>
    <presto-choice>
        Bronze Plan
        <span slot="description">$9.99 per month</span>
    </presto-choice>
</presto-choices>
`}

  </DemoTabs.Code>
    <presto-choices>
        <presto-choice>
            Gold Plan
            <span slot="description">$9.99, then $49.99 per month</span>
        </presto-choice>
        <presto-choice checked>Silver Plan
            <span slot="description">$39.99 per month</span>
        </presto-choice>
        <presto-choice>Bronze Plan
            <span slot="description">$9.99 per month</span>
        </presto-choice>
    </presto-choices>
</DemoTabs>

## Checkbox Type

In addition to a radio style selector, you can also choose a checkbox. This
lets you change the behavior from selecting one option from many, to selecting
any number of options, or none at all.

<DemoTabs>
  <DemoTabs.Code>
{`
<presto-choices>
    <presto-choice type="checkbox">
        Gold Plan
        <span slot="description">$9.99, then $49.99 per month</span>
    </presto-choice>
    <presto-choice checked>
        Silver Plan
        <span slot="description">$39.99 per month</span>
    </presto-choice>
    <presto-choice type="checkbox">
        Bronze Plan
        <span slot="description">$9.99 per month</span>
    </presto-choice>
</presto-choices>
`}

  </DemoTabs.Code>
    <presto-choices>
        <presto-choice type="checkbox">
            Gold Plan
            <span slot="description">$9.99, then $49.99 per month</span>
        </presto-choice>
        <presto-choice type="checkbox" checked>Silver Plan
            <span slot="description">$39.99 per month</span>
        </presto-choice>
        <presto-choice type="checkbox">Bronze Plan
            <span slot="description">$9.99 per month</span>
        </presto-choice>
    </presto-choices>
</DemoTabs>
