# Skeleton

```html
<ce-skeleton></ce-skeleton>
```

#### Skeletons are use to show the loading state of UI elements that will eventually be redrawn.

Skeletons are purposly simple and minimal to give you the most flexibility to designing layouts. You'll likely use more than one skeleton to achieve your desired effect.

## Properties

| Property | Attribute | Description      | Type                         | Default   |
| -------- | --------- | ---------------- | ---------------------------- | --------- |
| `effect` | `effect`  | Animation effect | `"none"` `"pulse"` `"sheen"` | `'sheen'` |

## Shadow Parts

| Part          | Description        |
| ------------- | ------------------ |
| `"base"`      | Component base div |
| `"indicator"` | Indicator div      |

## CSS Custom Properties

| Name              | Description                                                |
| ----------------- | ---------------------------------------------------------- |
| `--border-radius` | The skeleton's border radius.                              |
| `--color`         | The color of the skeleton.                                 |
| `--sheen-color`   | The sheen color when the skeleton is in its loading state. |
