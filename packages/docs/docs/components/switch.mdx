# Switch

```html
<ce-switch></ce-switch>
```

import DemoTabs from '../../components/Demo-Tabs.js';

<DemoTabs>
	<DemoTabs.Code>{ '<ce-switch>Toggle Me</ce-switch>' }</DemoTabs.Code>
	<ce-switch>Toggle Me</ce-switch>
</DemoTabs>

## Disabled

Use the `disabled` property to disable the switch

<DemoTabs>
	<DemoTabs.Code>
		{ '<ce-switch disabled>Disabled</ce-switch>' }
	</DemoTabs.Code>
	<ce-switch disabled>Disabled</ce-switch>
</DemoTabs>

## Custom Size

Use the `disabled` property to disable the switch

<DemoTabs>
	<DemoTabs.Code>
		{
			'<ce-switch style="--width: 80px; --height: 32px; --thumb-size: 26px;"></ce-switch>'
		}
	</DemoTabs.Code>
	<ce-switch
		style={ {
			'--width': '80px',
			'--height': '32px',
			'--thumb-size': '26px',
		} }
	></ce-switch>
</DemoTabs>
