## Disabled

Use the `disabled` property to disable the switch

<DemoTabs>
	<DemoTabs.Code>
		{ '<sc-switch disabled>Disabled</sc-switch>' }
	</DemoTabs.Code>
	<sc-switch disabled>Disabled</sc-switch>
</DemoTabs>

## Custom Size

Use the css variable design tokens to change the sizes of the switch for an instance.

<DemoTabs>
	<DemoTabs.Code>
		{
			'<sc-switch style="--width: 80px; --height: 32px; --thumb-size: 26px;"></sc-switch>'
		}
	</DemoTabs.Code>
	<sc-switch
		style={ {
			'--width': '80px',
			'--height': '32px',
			'--thumb-size': '26px',
		} }
	></sc-switch>
</DemoTabs>
