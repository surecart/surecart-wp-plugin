# Dropdown

```html
<ce-dropdown></ce-dropdown>
```

Dropdown is a component that opens a floating content panel when a trigger element is clicked.
This components takes care of updating the state of the dropdown menu (opened/closed) as well as handles
closing the menu when clicking outside.
